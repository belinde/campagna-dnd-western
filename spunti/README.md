# Spunti

Questa cartella raccoglie idee, agganci narrativi e potenziali eventi futuri per la campagna. I contenuti qui presenti **non sono parte dell'ambientazione canonica**: descrivono cose che *potrebbero* accadere, non cose che sono già accadute.

## A cosa servono

Gli spunti sono uno strumento del DM per tenere traccia di idee da sviluppare: una fazione che potrebbe entrare in scena, un PNG che potrebbe comparire, un evento che potrebbe scattare se i PG fanno certe scelte. La modalità **ingame** li consulta come fonte di ispirazione durante la sessione.

## Quando rimuovere uno spunto

Quando uno spunto diventa realtà — perché è stato giocato e trascritto nel resoconto — il file può essere eliminato o spostato nell'ambientazione canonica appropriata. La decisione spetta al DM.

## Convenzione nomi file

Nomi liberi, senza prefisso di tipo:

```
spunti/nome-idea.md
```

Esempi: `spunti/rivolta-halfling.md`, `spunti/ritorno-blackwood.md`, `spunti/treno-rapinato.md`

## Struttura consigliata per un file di spunto

```markdown
# Titolo dello spunto

**Tipo:** evento / PNG / luogo / fazione / altro

## Descrizione

Breve descrizione dell'idea: cosa potrebbe succedere, chi è coinvolto, in quale contesto.

## Agganci narrativi

Come potrebbe collegarsi alla situazione attuale dei PG o agli eventi in corso.

## Note DM

Considerazioni riservate: quando introdurlo, condizioni che lo attivano, alternative possibili.
```
