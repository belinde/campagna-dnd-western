# Importa immagine

Prima di qualsiasi azione, leggere con `Read` il file `.cursor/skills/campagna-immagini/SKILL.md` e applicare **solo** la sezione **«Import asset immagine»** (ignorare la sezione Prompt).

Richiede modalità Agent con permessi di scrittura su file e asset. La rule `campagna` resta il contesto base del progetto.
